# Reorganized documentation

This package keeps the project documentation but reduces operational bloat.

Files:

```text
HANDOFF.md
  Short restart file. Current state and next actions only.

docs/CANON.md
  Short canon. Mandatory principles only.

docs/METHOD_APPENDIX.md
  Detailed methods, candidates, and prototype structures.

docs/HISTORY.md
  Decision history. Includes H-057 for this slimming step.

docs/archive/
  Full pre-slim CANON and HANDOFF for traceability.
```


## WSA discipline update

This package incorporates a tightening pass based on WSA operating discipline:

```text
- Keep CANON slim.
- Freeze speculative method candidates in METHOD_APPENDIX.
- Prefer third-company case addition over further specification.
- Treat lint as the first enforcement mechanism.
- Distinguish unknown, no public trace, unassessed, out-of-scope, and pending.
- Permit honest asymmetry between compared firms.
```
